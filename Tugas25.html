Mengenal Semantic Elements Di HTML

  • Article

Ketika kita ingin membuat layout utuk menampilkan block artikel-artikel yang ada di website kita, tag artikel bisa kita gunakan dengan element header dan footer.
Karena tag artikel memiliki type block, artinya element ini memiliki lebar sesuai dengan lebar browser jika kita tidak menentukan lebarnya.

  • Aside

Elements ini bisa kita gunakan untuk melengkapi layout utama. Misal kita mempunyai halaman blog maka kita bisa menambahkan aside dibagian bawah postingan untuk menjelaskan identitas penulis artikel. Karena aside memiliki type/sifat block maka lebarnya sesuai page jika kita tidak menentukan lebarnya.
